Download & install

How to get DocuFill

Follow these steps to download and install the extension in Chrome. The whole process takes about a minute.

1
Download the extension
Click the blue Download DocuFill button below. Your browser will download a file named docufill-extension.zip.
2
Unzip the file
Locate the downloaded zip on your computer (usually in your Downloads folder). Right‑click it and choose Extract all (Windows) or double‑click to unzip (Mac). Save the extracted folder somewhere you’ll keep it (e.g. Documents\DocuFill). You’ll need this folder for the next step.
3
Open Chrome’s Extensions page
In Chrome, type chrome://extensions in the address bar and press Enter. Or open the menu (⋮) → Extensions → Manage Extensions.
4
Turn on Developer mode
On the top‑right of the Extensions page, toggle Developer mode to ON.
5
Load the extension
Click the Load unpacked button. A folder picker will open. Select the folder you extracted in step 2 (the one that contains manifest.json). Click Select Folder. DocuFill will appear in your extensions list and the icon will show in your Chrome toolbar.

You’re done. Open a DocuSign document, click the DocuFill icon in the toolbar, and follow the usage guide below to upload your data and fill forms.

Usage guide

How to use DocuFill

Step-by-step instructions for loading your data, mapping fields, and autofilling DocuSign documents.

1. Open a DocuSign document

Go to docusign.com (or docusign.net), sign in, and open an envelope or document where you can fill in fields. Keep this tab open.

2. Open the DocuFill popup

Click the DocuFill icon in your Chrome toolbar (you may need to click the puzzle piece and pin DocuFill). The extension popup will open.

3. Upload your data file

In the popup, click Upload CSV/Excel and choose a .csv, .xlsx, or .xls file. The first row should be your column headers (e.g. Name, Email, Company). DocuFill will show your columns and a list of records (rows).

4. Map DocuSign fields to your columns

Click Start mapping (or Map fields). Then click on a field in the DocuSign document. DocuFill will ask you to choose which column from your file should fill that field. Select the matching column and confirm. Repeat for every field you want to autofill. When done, click Stop mapping. You can save this mapping as a template so you don’t have to map again for the same document type.

5. Fill the document

Select the record (row) you want to fill from the dropdown or list. Click Fill (or Fill form). DocuFill will fill all mapped fields with that record’s data. Use Next record to move to the next row and fill again, or change the selection and fill with a different record.

Tips

  • Use Export template to save your mapping; use Import template to load it on another computer or after reinstalling.
  • If you have multiple sheets in an Excel file, use the sheet picker to choose which sheet to use.
  • Search the records list by typing in the search box to find a specific row quickly.

Privacy & security

Your data stays yours

DocuFill does not send your data to any server. No signup, no tracking, no cloud upload.

  • All data stays local — Your CSV and templates are stored in Chrome’s local storage on your device.
  • No server uploads — Nothing is sent to DocuFill or any third party.
  • Extension permissions — DocuFill only requests storage, active tab, and access to DocuSign domains. It cannot read other websites.
